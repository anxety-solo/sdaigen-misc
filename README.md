# sdAIgen-misc

Small user scripts for Stable Diffusion WebUI | *Written exclusively for [sdAIgen](https://github.com/anxety-solo/sdAIgen)~ ;33*

## Files

- [`add_githublink.js`](./add_githublink.js)
> Adds a link to GitHub in the footer of the WebUI interface.

- [`remove_hypernetwork.js`](./remove_hypernetwork.js)
> Removes the **Hypernetworks** tab from the interface.

- [`webui_timer.js`](./webui_timer.js)
> Session timer + notification buttons, NSFW blur, and quick jump to CivitAI.
